Notice Explicative - inventaire

L'inventaire reprend l'intégralité du stock à la clôture de l'exercice et permet de valoriser le stock. Chaque entrée reprend l'ensemble les informations nécessaires pour une desune écriture et comprend plusieurs élément marchandises qui compose le stock invendu :

Rubriques :

  • Description : La description de l'article.
  • Numéro : Numéro unique de l'article.
  • Libellé : Quantité présente dans le stock.
  • Valeur unique : Valeur pondérée pour un article.
  • Valeur totale : Valeur totale pour l'ensembles des articles présents.
