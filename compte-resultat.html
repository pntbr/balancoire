Notice Explicative - Compte de Résultat

Le compte de résultat est un document comptable qui présente les produits et les charges de l'association sur une période donnée, permettant de déterminer le résultat net (bénéfice ou perte) de l'association. Voici les rubriques principales :

Produits d'activité :

Cotisations des membres : Les contributions versées par les membres de l'organisation.

Dons : Les contributions financières reçues sous forme de dons.

Prestations de services : Les revenus générés par la fourniture de services.

Ventes de produits : Les revenus générés par la vente de produits.

Autres produits : Collecte le reste des revenus générés.

Total des produits de l'activité : La somme de toutes les sources de revenus de l'activité.

Charges de l'activité :

Achats de marchandises : Le coût des marchandises destinées à être vendues.

Achats d'approvisionnements : Le coût des matières premières, des fournitures et des services utilisées dans l'activité.

Variations de stocks : La valorisation du stock existant à la clôture.

Charges externes : Les dépenses externes diverses comme l'assurance, le loyer, etc.

Impôts, taxes et versements assimilés : Les charges fiscales et les taxes liées à l'activité.

Autres charges : Les dépenses diverses non incluses dans les autres catégories.

Total des charges de l'activité : La somme de toutes les charges d'activité.

Résultat courant avant impôts :

La différence entre les produits de l'activité et les charges.

Impôt sur les bénéfices :

Les taxes dues sur le résultat courant avant impôts.

Résultat net :

L'excédent ou le déficit après déduction de l'impôt sur les "sociétés".
